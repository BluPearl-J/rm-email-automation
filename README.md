# Smart CRM Email Follow-Up Automation
Built by Joy O. Reuben-Atohor

Fully automated CRM follow-up system using Google Sheets + Apps Script.
Sends personalised email sequences, flags overdue contacts, and delivers daily digests.

## Features
- Automated email follow-up every 3 days
- First, second, and final follow-up sequences
- Daily digest to admin
- Google Calendar reminders for high-priority contacts
- Colour-coded status tracking

## Built with
Google Sheets · Apps Script · Gmail API · Google Calendar API
